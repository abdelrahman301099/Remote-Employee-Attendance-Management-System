
NetBlaze Solution:

A simple full-stack template solution using Blazor Web App (Auto rendering mode) for the frontend and ASP.NET Core Web API for the backend.

✨ Features

✅ Blazor Web App with Auto rendering mode
✅ ASP.NET Core RESTful API backend
✅ Shared project for common DTOs, Enums, Utilities, etc.
✅ Ready for local development and deployment
✅ Simple and clean architecture
